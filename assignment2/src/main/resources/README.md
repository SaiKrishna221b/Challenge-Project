# Resources

This directory contains the static data files required by the application.

## Files

### `sales_data.csv`
*   **Description**: A generated dataset of sales transactions.
*   **Size**: ~20,000 rows.
*   **Schema**: `TransactionId,Date,Category,Product,Region,Quantity,UnitPrice`
*   **Purpose**: Used by `SalesAnalyzer` to demonstrate parsing and aggregation logic.
*   **Generation**: This file was programmatically generated to ensure a diverse distribution of categories and regions for meaningful testing.
